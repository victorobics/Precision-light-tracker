# Precision-light-tracker
Here you can get all the code and files needed for the precision ight tracker project and also the steps to do it

CAD design of the structure:

I make the CAD design of the project in order to be able to print them, and I ensure that all the parts fit in. Moreover I separate the project in 3 parts, it contains the base where the Arduino nano goes and also the servo that moves to the sides. The second part is where the second servo goes wich moves up down the final part that is the structure that contains the 4 sensors and track the light.
